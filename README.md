# プライズネス健康経営サポート LP

企業・従業員向けの健康経営サポート用1枚LPです。

## ファイル構成

- `index.html`: ページ本文
- `styles.css`: レイアウトとデザイン
- `script.js`: スマホメニューと問い合わせ文面作成
- `vercel.json`: Vercel用の静的サイト設定

## 公開前に差し替えるもの

1. 必要であれば、ヒーロー画像
   - 現在は外部画像URLを使用しています。
2. 本体サイトからの導線
   - 本体サイト側に「法人向け健康経営サポートはこちら」を設置し、このLPへリンクします。

問い合わせ先メールアドレスは `mail@prizenes.com` に設定済みです。

## Vercel

Vercelでは静的サイトとしてそのままデプロイできます。ビルドコマンドは不要です。
